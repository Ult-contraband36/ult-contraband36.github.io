📋 About the Project

Glyph brings server management to your desktop. It provides a clean interface for you to monitor and control remote machines. You no longer need to switch between different tools or use complex command-line windows for routine tasks. Glyph combines essential server actions into one smooth workspace.

✨ Core Features

  • Unified Dashboard: View all your server statuses from one screen.
  • File Management: Move and edit files on remote servers using a simple interface.
  • One-Click Connections: Save your server details and access them whenever you need.
  • Modern Interface: Navigate through settings and logs with tools built for speed and clarity.

⚙️ System Requirements

Glyph runs on modern versions of Windows. Ensure your computer meets these items for the best experience:

  • Windows 10 or Windows 11 (64-bit).
  • At least 4GB of RAM.
  • A working internet connection for server communication.

🛡️ Security and Privacy

Glyph respects your data. The application stores your connection details locally on your machine. We do not transmit your login credentials or server addresses to outside servers. You maintain full control over your data at all times. Because Glyph operates locally, your privacy remains secure while you work.

💡 Frequently Asked Questions

Can I connect to multiple servers? Yes, you can add and manage as many servers as you need within the application.

Does it support file uploads? Yes, the interface includes a feature to upload and download files between your local computer and your remote server.

Is there a limit on server connections? No, you decide how many connections you want to keep active in your dashboard.

🛠️ Support

If you encounter issues during installation, check your network settings first. Most connection problems stem from firewalls or incorrect login credentials. If problems persist, ensure your server allows SSH connections, as Glyph uses this method to communicate with your hardware.
